• Entropy Card Game

    • What is this game?

    This is basically a level up version of the traditional rock,paper,scissor game with a card element "Rock, Paper, Scissors, Lizard, Spock," made famous by "The Big Bang Theory," adds Lizard and Spock to the classic game. Unlike regular Rock, Paper, Scissors, where each thing beats one and loses to another, this version has each choice interacting with more than just one other choice. This makes the game more strategic, as players have to think about more possibilities with each move they make.
